- `willbuddy-facebook-cover.svg` - 1640x624 Facebook page cover source.

## Visual Tokens

- Background: `#FAF8F5`
- Surface: `#F0EBE4`
- Border: `#E8E0D6`
- Accent green: `#5B7A5E`
- Text: `#2D2A26`
- Muted text: `#5B4F3E`
- Secondary muted text: `#7A6F62`
- Typeface direction: Libre Baskerville or Georgia for brand headings, system sans for UI copy.

## Usage

Use the PNG files for platform upload. Keep the SVG files as editable sources. Do not post publicly from these assets until Miguel approves the specific profile or post action.
